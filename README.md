# Star Wars Survey 

This project aims to perform analysis of star wars movies. Using the input file below, we find favorite star war movie, most and least favorable character, relation between rank and total views of the movie etc. The analysis of the movies can be found [here](https://github.com/siddhantj/StarWarsSurvey/tree/master/output). The input file is available [here](https://github.com/siddhantj/StarWarsSurvey/blob/master/data_dir/star_wars.csv)
